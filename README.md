# Basic HTML Website
This is a simple mulit-page website built using **semantic HTML and basic CSS** as part of the roadmap.sh Frontend projects.

The project focuses on correct HTML structure, navigation between pages, SEO meta tags, and  a contact form.

## pages Inculded
Home
About
Projects
Contact

## Features
Semantic HTML structure
Navigation bar on all pages
SEO meta tags on each page
Contact form with name, email, and message fields
Simple and clean layout using CSS
